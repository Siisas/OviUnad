  SQL Server Integration Services (SSIS) es la herramienta de integración y ETL (extraer - transformar - cargar) en la pila de la plataforma de datos de Microsoft. SSIS se usa normalmente en escenarios de almacenamiento de datos, pero también se puede usar en casos comunes de uso de integración de datos o simplemente para mover datos. SSIS se utiliza entre bastidores en los Planes de mantenimiento de SQL Server y en el Asistente de importación / exportación. SSIS es una herramienta flexible y puede realizar todo tipo de operaciones: Transferencia de datos desde una fuente a un destino. Esto se hace en la memoria y puede realizar tareas de manipulación de datos en los datos mientras está en la memoria, lo que hace de SSIS una de las herramientas más rápidas del mercado. Puede realizar tareas sencillas de FTP. SSIS puede enviar correos electrónicos para notificar a las personas. SSIS es capaz de un robusto manejo de errores y eventos. Puede definir un flujo de trabajo con restricciones para ejecutar condicionalmente ciertas tareas. Y si todo eso no es suficiente, siempre puede extender SSIS con código .NET.
